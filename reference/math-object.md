### Most useful methods

- `Math.pow()`
- `Math.sqrt()`
- `Math.round()`
- `Math.ceil()`
- `Math.floor()`
- `Math.abs()`
- `Math.max()`
- `Math.min()`

### Examples

    Math.pow(2) // 4
    Math.sqrt(27) // 3
    Math.round(42.4) // 42
    Math.ceil(7.004) // 8
    Math.floor(45.95) // 45
    Math.abs(-2) // 2
    Math.max(-10, -20) // -10
    Math.max(-10, -20) // -20
